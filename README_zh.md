# TripWireDisarmer
[English](README.md) | **简体中文**

允许您使用剪刀制作出带有 disarmed + powered 属性的绊线，可用于在 NeoForge 中制作绊线复制机（刷线机）。

### 特性

* 本模组不会对 NeoForge 已有修改（左键点击绊线直接拆除绊线）进行改动，通过为剪刀附加额外的操作来实现原版行为；
* 对于制作特殊绊线，尽可能模拟原版行为；

### 使用方法

1. 安装本模组；
2. 手持剪刀（或包含 `#c:tools/shear` 标签的物品）并 **右键点击** 已连接的绊线；
3. 大功告成！

### 安装说明

本模组必须在服务端（或单人模式）安装，客户端可选安装，下方列表给出了几种不同安装方式的现象

| 服务端 | 客户端 | 现象                     |
|-----|-----|------------------------|
| √   | √   | 模组可以正常工作,所有功能工作良好      |
| √   | ×   | 玩家可以通过右键改变绊线状态,但手部没有动作 |
| ×   | √   | 仅在单人模式下有效果             |
| ×   | ×   | 无效果                    |
